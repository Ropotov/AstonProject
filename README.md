# Rick and morty
Приложение написано на Java и Kotlin.

API https://rickandmortyapi.com/

## Стек

* Kotlin
* Java
* Retrofit
* MVVM, Clean architecture
* Coroutines
* RxJava
* Pagging3
* Room
* Dagger2
* Glide

## Информация

Приложения отображает списки персонажей, локаций и эпизодов мультсериала Rick and Morty.
